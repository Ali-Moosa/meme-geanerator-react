# Meme Generator

A modern React application that allows users to create custom memes by adding top and bottom text to randomly fetched images from the Imgflip API. The app features a stunning purple gradient UI with smooth animations, responsive design, and real-time text editing capabilities to generate hilarious memes on demand.
